유저 어뷰징 성향 지표 고도화:
광고 수익 기여도 기반 정량 지표 설계

단순 행동 지표를 넘어 광고 수익 기여도를 반영한 유저 어뷰징 성향 지표를 설계하고, 다수의 서비스에 공통 적용 가능한 구조로 고도화한 프로젝트

배경

기존에는 유저의 어뷰징 성향을 단순 포인트 획득량 대비 광고 노출 수로 측정하였다. 그러나 이 방식은 광고 유형별 수익 기여도 차이를 반영하지 못하고, 서비스 간 비교가 어렵다는 한계가 있었다.

이에 광고 수익 기여도를 분모에 반영한 새로운 지표 구조를 설계하고, 국내외 여러 서비스에 공통 적용 가능한 형태로 고도화하는 프로젝트에 참여하였다.

광고 지면별 실제 단가를 직접 구할 수 없는 환경에서, 집계 데이터 기반 수익 추정 방식을 설계하는 것이 핵심 과제였다. 지면별 단가 구조를 분석하고 수익 proxy를 설계하여 분모에 반영하였다.

고도화 단계

1
기존 지표 검증
기존 지표가 실제 어뷰징 발생 시점과 얼마나 일치하는지 교차 검증하여, 간접 모니터링 지표로서의 활용 가능성을 확인하였다.
2
분모 고도화 — 광고 수익 기여도 반영
광고 지면별 수익 구조를 분석하여, 유저별 실제 수익 데이터가 없는 경우 집계 데이터 기반 수익 proxy를 설계하였다. 지면별 단가 차이를 가중치로 반영하여 수익 기여도를 추정하였다.
3
분자 고도화 — 포인트 유형별 어뷰징 신호 분석
포인트 획득 유형별로 어뷰저와 정상 유저 간 분포 차이를 분석하여, 어뷰징 신호 강도가 높은 유형을 식별하였다. 통계적 검정을 통해 유형별 가중치 설계 근거를 마련하였다.
4
환경 보정
OS별 환경 차이로 인한 지표 왜곡 가능성을 검토하고, 유의미한 차이가 확인될 경우 보정 방안을 적용하는 단계를 설계하였다.

주요 작업

  • 1
    광고 수익 구조 분석 — 광고 지면별 수익 데이터 존재 여부와 집계 방식을 분석하여, 유저별 수익 기여도를 추정하는 4가지 케이스로 분류하고 각각의 계산 방식을 설계하였다.
  • 2
    수익 proxy 설계 — 광고 지면별 집계 수익 데이터를 활용하여 지면별 평균 단가를 산출하고, 유저별 노출 수에 단가를 곱하는 방식으로 수익 기여도 proxy를 구현하였다.
  • 3
    포인트 유형별 어뷰징 신호 분석 — 어뷰저와 정상 유저 간 포인트 획득 패턴 차이를 분석하여, 어뷰징 신호 강도가 높은 유형과 낮은 유형을 구분하였다. Mann-Whitney U 검정 등 통계적 방법론을 적용하였다.
  • 4
    다수 서비스 적용 설계 — 서비스마다 광고 구조와 테이블 구성이 다름을 고려하여, 공통 적용 가능한 지표 프레임워크를 설계하고 서비스별 구현 방향을 정리하였다.

시사점

데이터 제약 환경에서의 지표 설계

유저별 실제 광고 수익 데이터에 접근하기 어려운 환경에서, 집계 수준의 데이터를 활용하여 현실적인 수익 proxy를 설계하였다. 완벽한 데이터가 없는 상황에서 어떤 가정을 세우고 어떤 근사 방식을 선택할지 판단하는 경험을 쌓았다.

공통 프레임워크 설계의 어려움

서비스마다 광고 구조와 데이터 환경이 달라, 단일 공식을 모든 서비스에 그대로 적용하기 어렵다는 점을 확인하였다. 해석 기준은 통일하되 서비스별 구현 방식은 유연하게 가져가는 방향으로 설계하였다.
